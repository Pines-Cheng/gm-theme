###使用
请确定已经安装nodejs及npm.

请先运行命令:`npm install`安装依赖.

### 发布文档
将docs下的文档发布到gh-pages分支:

```
➜  gm-theme git:(master) ✗ npm run deploy
> gm-theme@1.0.0 deploy /Users/spider/Work/gm_lib/gm-theme
> gh-pages -d docs
```
